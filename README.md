# Getting started

- Clone repo
- Fill out Server/appsettings.Development.json
  - Cloudinary
  - AppSettings
  - ConnectionStrings
- run `make migrations`
- run `make frontend`
- run `make backend`

# Enviroment Variables

- CLOUDINARY_NAME
- DB_CONN_STR
- DB_PASSWORD
- APPSETTINGS_TOKEN
- CLOUDINARY_APIKEY
- CLOUDINARY_APISECRET
